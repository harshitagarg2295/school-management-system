# 🏫 VidhyaLitee - Digital Multi-School Management ERP

VidhyaLitee is a modern, intuitive, and all-in-one School Management ERP software designed to digitalise educational institutions. It automates daily administrative tasks, streamlines fee collection, and transforms schools into a completely paperless system.

---

## 🚀 Key Features

*   **Multi-School Architecture:** Manage multiple schools or branches efficiently from a centralized dashboard.
*   **Smart Fee Payment:** Integrated with **Razorpay** for seamless, secure online fee collection with automated digital receipts.
*   **Attendance Tracking:** Real-time attendance management for students and staff with detailed monthly reports.
*   **Instant Notices & Circulars:** Send important announcements, events, and digital circulars to parents and staff instantly.
*   **Student & Staff Management:** Comprehensive database management for tracking student progress, academic records, and staff details.

---

## 🛠️ Tech Stack

*   **Backend:** Node.js, Express.js
*   **Frontend:** HTML5, CSS3 (Inter UI, FontAwesome), JavaScript / EJS
*   **Database:** MongoDB (or whichever database you are using)
*   **Payment Gateway:** Razorpay API
*   **Deployment:** Render
